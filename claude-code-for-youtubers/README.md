# Claude Code for YouTubers

Turn Claude Code into your YouTube production assistant. System prompt + 7-stage content pipeline.

**Companion video:** [VIDEO_LINK]

## What's in here

- `CLAUDE.md` — Drop-in system prompt for Claude Code. Covers all 7 stages of the content pipeline: strategy, ideation, production, publishing, measurement, learning, and looping. Customize it for your channel.

## How to use

1. Copy `CLAUDE.md` into your project root
2. Replace the `[customize these]` fields with your channel info
3. Open Claude Code in that directory
4. Start with: "What should I record today?"

## The 7 stages

1. **Strategy** — Channel rules, audience, what works / what doesn't
2. **Ideate** — Research topics, check competition, find first-mover windows
3. **Produce** — Generate titles, thumbnail briefs, hooks, full packaging
4. **Publish** — Metadata, descriptions, cross-platform distribution
5. **Measure** — A/B test titles, analyze comments and transcripts
6. **Learn** — Log what worked, log what didn't
7. **Loop** — System reads the learnings next session and gets smarter
